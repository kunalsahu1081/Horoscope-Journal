##  Quick Start Commands for android

```bash
# Install
npm install

# Run on Android
npm start          # Terminal 1
npm run android    # Terminal 2

# Clean build
cd android && ./gradlew clean && cd ..

# Reset cache
npm start -- --reset-cache

# Build release APK
cd android && ./gradlew assembleRelease
```
